# Events NextJs App

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## About App
Demo Application that handles filtering and showing events and events details!

## Built Using
- NextJs 
- ReactJs
- CSS Module
